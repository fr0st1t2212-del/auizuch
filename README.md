Представляю вашему вниманию Rova Launcher — легковесный лаунчер для горизонтальной установки игровых модификаций, скриптов и пресетов. Основная задача проекта — сокращение жизни игрока: больше не нужно вручную копаться в архивах и переносить файлы. Выбрал версию, нажал «Установить» — и всё готово.

Основные функции:

• Автоматический поиск игры: Лаунчер сам сканирует диски на наличие gta_sa.exe.

• Выбор варианта: Поддержка нескольких конфигураций/скриптов через облачный JSON-конфиг.

• Новости: Встроенная лента обновлений прямо в интерфейсе.

• Discord RPC: Отображение вашего воздействия в Discord.

• Связь с разработчиком: Возможность отправить фидбек или сообщение «Нужно прямо из лаунчера».

Инструкция:

  1. Запустите лаунчер.

  2. Авторизуйтесь (для теста: логин user, пароль user).

  3. Укажите путь к файлу с игрой (если не найден автоматически).

  4. Выберите нужный патч/скрипт в выпадающем списке.

  5. Нажмите «УСТАНОВИТЬ» и дождитесь завершения загрузки.

Технические подробности:

- Язык: Python (CustomTkinter)

- Версия: 1.0

- Зависимости: Работает автономно (скомпилирован в .exe)
